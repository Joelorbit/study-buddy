Studdy Buddy Landing Page

Studdy Buddy is an AI-powered study assistant landing page designed to showcase features like personalized study plans, AI-generated summaries & quizzes, daily streak tracking, and more. Built with Svelte and GSAP animations.

Features

Hero section with catchy tagline and call-to-action button

Feature cards highlighting key functionalities

3D rotating cube animation showcasing core features

Subscription call-to-action with email input

Responsive design for all devices

Technologies

Svelte – frontend framework

GSAP – animations and 3D cube

CSS / Flexbox / Grid – layout and styling

Run Locally
# Install dependencies
npm install

# Start development server
npm run dev
